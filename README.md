# 写真展 POS システム

Google Apps Script（GAS）で動作する写真展向け簡易POSアプリです。

## ファイル構成

```
photo-pos/
├── index.html   # POSアプリ本体（GASから配信）
├── Code.gs      # GASバックエンド
└── README.md    # このファイル
```

## セットアップ手順

### 1. Googleスプレッドシートを作成

新しいスプレッドシートを作成し、名前を「写真展POS」などにします。

### 2. Apps Script を開く

スプレッドシートのメニュー →「拡張機能」→「Apps Script」

### 3. ファイルを配置

**Code.gs**
- デフォルトの `コード.gs` の中身を `Code.gs` の内容に置き換えて保存

**index.html**
- Apps Script エディタの左側「＋ ファイルを追加」→「HTML」
- ファイル名を `index`（拡張子なし）にする
- `index.html` の内容を貼り付けて保存

### 4. Webアプリとしてデプロイ

1. 右上「デプロイ」→「新しいデプロイ」
2. 種類：**Webアプリ**
3. 次のユーザーとして実行：**自分**
4. アクセスできるユーザー：**全員** ← ここが重要
5. 「デプロイ」をクリック
6. 権限の確認ダイアログ → 許可
7. 発行された URL をコピー（`https://script.google.com/macros/s/xxxxx/exec` 形式）

### 5. URLをブックマーク

コピーしたURLをiPadのSafariで開き、「ホーム画面に追加」すればアプリ感覚で起動できます。

## シート構成（自動生成）

| シート名 | 内容 |
|---------|------|
| 売上明細 | 精算のたびに自動追記 |
| 商品マスター | 商品・チケット一覧（アプリから編集可） |
| 担当者マスター | 担当者一覧（アプリから編集可） |

---

## 商品・担当者の変更方法

アプリを開いた状態で：

- 「📦 商品」タブ → 編集 →「💾 スプレッドシートに保存」
- 「👥 担当者」タブ → 編集 →「💾 スプレッドシートに保存」

次回起動時にスプレッドシートから最新データを自動読み込みします。

---

## GASコードを更新した場合

コードを変更したら必ず「新しいバージョンとしてデプロイ」してください。
（「デプロイを管理」→「編集」→バージョン「新しいバージョン」）

URLは変わりません。

---

## 注意事項

- インターネット接続が必要です
- オフライン時はローカルキャッシュで動作しますが、スプレッドシートへの記録はできません
- Googleアカウントが必要です（ログイン不要で使えます）
